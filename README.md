# Formulário Completo
Este projeto tem como propósito ajudar pessoas que trabalham com formulários, assim como advogados, contadores entre outros

## Tecnologias usadas
- HTML
- CSS
- JavaScript

## Funcionalidades
- ❇️ Vamos preecher de forma manual iniciamente
- ❇️ Vamos usar os mesmos dados para estar criando outros tipos de documento
- ❇️ Vamos criar declarações, currículos e até atencedentes prontos.

## Como rodar
1. Clone o repositório
2. Abra o index.html no navegador
